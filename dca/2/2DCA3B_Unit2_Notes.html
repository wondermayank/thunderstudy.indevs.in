Introduction to Adobe PageMaker
▾
📘 Definition Adobe PageMaker is the "world's leading cross-platform professional page layout software." PageMaker is primarily used for designing and producing publications that require a combination of text and graphics.

PageMaker has a rich array of facilities to import text and artwork from other computer application packages, as well as allowing you to generate these directly from within PageMaker itself. PageMaker can handle text better than Illustrator and Photoshop and also gives you the flexibility of graphic control not available in word processors.

Key Capabilities

📄
Page Layout
Professional page layout for publications
🖼
Text + Graphics
Combines text and graphics flexibly
📥
Import/Export
Imports from many application packages

Aldus & Adobe PageMaker
▾

PageMaker was the first desktop publishing program, introduced in 1985 by Aldus Corporation, initially for the Apple Macintosh but soon after also for the PC. It relies on Adobe Systems' PostScript page description language.

In 1994, Aldus and PageMaker were acquired by Adobe Systems. PageMaker was awarded a SPA Excellence in Software Award for Best New Use of a Computer in 1986.

📌 Key Dates Introduced: 1985 (by Aldus Corporation) | Acquired by Adobe: 1994 | Final Version: 7.0 (July 9, 2001)

Toolbox — Understanding PageMaker Tools
▾

The Toolbox contains all the tools used to create a publication in PageMaker. To display the toolbox: Window → Show Tools.

Zero Position
Allows setting the zero position on the screen. Zero position is where both the top and left side rulers are at zero. All measurements are made relative to this point. Click and drag to set it anywhere on the workspace.
Pointer Tool
Used to pick, drag and drop graphics and text.
Text Tool
Used to edit and select text and insert text frames. Click anywhere in the document and start entering text. Creates a free-flowing text box.
Rotate Tool
Used to rotate text and graphic objects. Click the rotate tool, click on the object reference point and drag to rotate.
Crop Tool
Used to crop imported graphic images to the required size. Works only on TIFF type images. Click and drag the image handles to crop.
Oblique Line Tool
Used to draw slanting lines. Click and drag to draw a line in any direction.
Constrained Line Tool
Draws lines at 45-degree intervals — horizontally, vertically, or diagonally.
Box Tool
Used to draw a box. Click and drag to draw a box in any direction.
Rectangle Frame
Used to type text inside a box. Draw the area using the tool, then select the Text Tool and click inside the frame to enter text. Text is constrained to the rectangular frame.
Circle Tool
Used to draw a circle or elliptical graphic. Select the tool, click and drag to draw a circle on the document.
Circular Frame
Used to type text inside a circle or oval. Draw the frame using Circular Frame Tool, then click inside with Text Tool to enter text.
Polygon Tool
Used to draw a shape with more than four sides. Modify by selecting Element → Polygon Settings.
Polygon Frame
Puts text inside a polygon. Select tool, draw a polygon, then select Text Tool and click inside the frame to enter text.
Hand Tool
Used to move the screen. Very useful when working with large documents. Select the tool and click and drag the page to view different parts.
Zoom Tool
Allows zooming in and out of the document view. Default is Zoom-in mode. To zoom out: hold Ctrl and click in the document window.

Styles Palette
▾

The Styles Palette is used to apply styles or to view or edit styles defined for the publication. To display: Window → Show Styles.

How to Apply a Style

  1. Click a paragraph with the Text Tool
  2. Click a style name in the palette
📌 Default Styles in Palette [No style], Body text, Caption, Hanging indent, Headline, Subhead 1, Subhead 2

Menu Bar Overview
▾

The standard menu bar contains all the main menus for File, Layout, Element and other utilities.

File Menu

CommandFunction
NewCreate a new publication — opens dialog box to define options before opening a new page.
OpenOpen an existing PageMaker file.
SaveSave the current publication. To save all open files, press Shift while choosing Save.
RevertRolls back all changes made since the last save. Same as closing without saving and reopening.
PlacePlace text and graphics into the publication.
AcquireScan an image into your publication.
ExportExport publications into the selected format.

Edit Menu

CommandFunction
Paste MultiplePaste multiple copies of text or graphics.
Paste SpecialSpecify the format to use when pasting an object into your publication.
Insert ObjectInsert an OLE embedded object within a PageMaker publication.
Edit StoryToggle between word processing mode and layout mode.

Layout Menu

CommandFunction
Sort PagesShows all publication pages in a dialog box; rearrange pages graphically.
Column GuidesDivide publication pages into different columns of equal size.
Auto FlowFlow text onto pages automatically from a loaded text icon. Flows text from column to column and adds pages based on text availability.

Type Menu

CommandFunction
Define StylesShows list of styles used in publication. Add, modify and delete styles.
FontChange the font of selected text.
SizeChange the point size of selected text.
LeadingAdjust line spacing.
AlignmentSet text alignment (Left, Center, Right, Justify).

Element Menu

CommandFunction
FrameOptions for frame alignment, position and margins.
Align ObjectsAlign multiple objects relative to each other.
Link OptionsDefine whether linked text or graphic is to be updated with the publication and its frequency.

Utilities Menu

Plug-ins: Contains a list of sub-commands listing various plug-ins included with PageMaker as additions.

View Menu

OptionFunction
Display Master ItemsToggle display of items from master pages to be reflected in current page of publication.
Display Non-Printing ItemsToggle display of non-printing items.

Window Menu

OptionFunction
Show ToolsHide/Show tools — toggles the PageMaker toolbar display.
Show Control PaletteShow/Hide Control Panel on screen.
Show ColorsShow/Hide Color Palette on screen.
Show StylesShow/Hide Styles Palette on screen.
Show LayersShow/Hide Layers Palette on screen.
Show Master PagesShow/Hide Master Pages Palette on screen.
Show HyperlinksShow/Hide Hyperlink Palette on screen.

Interface Elements

Ruler
Top and left ruler bars help precisely position cursor, text and graphics. Current cursor location shown by two dotted lines. View → Show Rulers.
Master Page Icon
Shows position of master pages. Clicking it displays the Master Page. Master Pages contain text, graphics, layout and settings common throughout the publication.
Page Icon
Shows list of pages in the publication; currently selected pages shown in black. Clicking a particular page icon displays that page.
Plasterboard
White space surrounding the page. Used to store graphics and texts while working. Items placed here are visible from any page and can be selected regardless of current page.

Page Layouts & Control Palette
▾

Creating a Document in PageMaker

After getting acquainted with PageMaker, you can create a publication (also called a Document). First, open a new blank document, then add text, pictures etc. according to your requirement. Thus a complete document is prepared which is printed on a printer.

Control Palette Overview

The Control Palette displays the properties for the selected paragraph, character or object. It is a shortcut to object, text, and paragraph properties. To display: Window → Show Control Palette.

Control Palette in Object Mode

When an object is selected, displays object properties:

PropertyFunction
Reference PointThe anchor point of the selected object
XY PositionSets horizontal and vertical position of the upper-left corner of the selected object
Width / HeightSets the width and height of the selected object (in inches or as percentage of current size)
RotateRotate the selected object around an axis. Center of rotation is set as object's center.
Horizontal/Vertical MirrorMirrors the selected object along the X or Y axis
ScalingScale the object proportionally
CroppingCrop the object

Control Palette in Character Mode

Displayed when the Text Tool is selected. Has options for changing text styles, size, leading, tracking, width, and kerning. A toggle button shifts between paragraph view and character view.

OptionFunction
KerningAdjusts spacing between specific characters. Selective — can adjust one or two characters. Negative numbers = closer together; Positive = further apart.
LeadingAdjusts spacing between lines of text. Applied line by line. Can be increased or decreased.
TrackingAdjusts how closely characters follow each other. Very tight = text closer together; Very loose = spreads out a bit.
SizeAdjusts point size of text.
PositionAdjusts placement on a line. Positive = superscript (above line); Negative = subscript (below line).
WidthAdjusts character width without affecting height. Over 100% = proportionately larger; Under 100% = proportionately smaller.

Control Palette in Paragraph Mode

Displays options for paragraph style, alignment, indents, spacing, grid spacing, and grid alignment.

Color Palette
▾

The Color Palette is used to apply colors or view the name or kind of color applied to the selected text or object. To display: Window → Show Colors.

How to Use Color Palette

  • The buttons Stroke, Fill, or Both can be used to change the color of the graphic object and object level.
  • Tints allow you to have variations of the same color.
  • To apply a color: select the text or object, then select the color from the Color Palette.

Color Modes

Stroke
Changes the outline/border color of a selected graphic object.
Fill
Changes the fill/inside color of a selected graphic object.
Both
Changes both stroke and fill color simultaneously.
Tints
Allows variations (lighter shades) of the same color to be created and applied.
✅ Summary of Key Palettes in PageMaker
  • Toolbox — All drawing, text, and editing tools (Window → Show Tools)
  • Styles Palette — Apply/view/edit text styles (Window → Show Styles)
  • Color Palette — Apply and manage colors (Window → Show Colors)
  • Control Palette — Object/character/paragraph properties (Window → Show Control Palette)
  • Layers Palette — Manage document layers (Window → Show Layers)
  • Master Pages Palette — Master page management (Window → Show Master Pages)
